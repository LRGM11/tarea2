Definición HTML5:

HTML: ( Lenguaje de etiquetas de hipertexto) es el componente más básico de la Web. Define el significado y la estructuradel contenido web. Ademas HTLM, generalmente se utilizan otras teconologias como es CCS y JavaScript.

Comó son los Elementos HTML

Los elementos son representados meidante etiquetas en un documento. Cada elemento aporta un siginificado semántico a su contenido.

Comó son los Atributos HTML

Son los valores o propiedades que añades a una etiqueta HTML. Este otorga una propiedad o comportamiento especifico a la propiedad etiqueta, por lo que es muy importante conocer de ambos.

Definicion CSS

Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

Comó son las Propiedades CSS

Abreviadas para agrupar la definición de propiedades comunes que actúan sobre el mismo tema.
